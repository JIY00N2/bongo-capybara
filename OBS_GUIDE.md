# OBS 연동 가이드

## 앱 실행
```bash
cd bongo-capybara
npm start
```

## OBS Window Capture 설정

1. OBS → 소스 → + → **윈도우 캡처** 추가
2. 윈도우: `[Electron] Bongo Capybara` 선택
3. **캡처 방식**: `윈도우 캡처 (BitBlt)` 또는 `Xcomposite` (Mac)
4. **커서 캡처**: 체크 해제
5. **소스 필터** → 색 제거 필터 추가:
   - 색상 공간: 사용자 지정
   - 키 색상: `#00FF00` (크로마키) 또는 투명 배경 그대로 사용

### Mac에서 창 투명 캡처
- OBS 버전 27+ 에서는 **Game Capture** 대신 **윈도우 캡처** 사용
- `투명 배경 허용` 체크 필요 (OBS 설정 → 고급 → 투명 소스 허용)
- 또는: 소스 → 필터 → **색 제거(크로마키)** 추가 후 배경색 제거

## macOS 접근성 권한

uiohook-napi가 글로벌 키 입력을 감지하려면 **접근성 권한**이 필요합니다.

1. 시스템 설정 → 개인 정보 보호 및 보안 → **접근성**
2. **터미널** (또는 앱을 실행한 앱) 체크 ✓
3. 앱 재실행

권한 없이도 앱은 실행되지만 키보드 후킹이 비활성화됩니다.

## 에셋 교체 방법

```
assets/
├── body/
│   └── body.png          # 카피바라 몸통 전체 (투명 배경 PNG)
├── hands/
│   ├── idle.png          # 기본 손 자세
│   ├── left.png          # 왼손 두드림
│   ├── right.png         # 오른손 두드림
│   └── both.png          # 양손 두드림
└── mouth/
    ├── closed.png        # 입 닫힘
    └── open.png          # 입 열림
```

- 모든 PNG는 **동일한 캔버스 크기** 권장 (예: 300×350px)
- 투명 배경 필수
- `body.png`가 존재하면 자동으로 PNG 모드로 전환됨
